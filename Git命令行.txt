# 全局配置
ssh-keygen -t rsa -C "你的github邮箱"                     # 生成 SSH 密钥


# 项目配置
git init                                                 # 初始化本地git仓库（仅第一次执行）

git add .                                                # 添加全部文件到暂存区
git commit -m "修改说明"                                  # 提交到本地仓库

git remote add origin ssh链接                            # 绑定远程仓库
git push   -u  origin main                               # 推送代码到 GitHub 

# 其他高频命令
git state                                                # 查看当前文件修改(红色=未加入暂存，绿色=已暂存待提交
git branch                                               # 查看所有分支
git branch   dev                                         # 创建新分支dev
git checkout dev                                         # 切换至dev分支



常见两种校验方式：
1. HTTPS + 账号密码（用户名 + PAT令牌）
2. SSH链接 + 密钥
